# Markdown写作笔记
——— markdown作为一种轻量级的标记语言，它允许人们使用纯文本的形式编辑文档，然后转化为html,非常适合用于撰写技术文档或者发布文章  
&nbsp;
## Markdown基本语法
标题: 以#开头，最后一个#后加空格，有几级#代表有几级标题（最高六级）
&nbsp;

换行：以空行隔开，markdown会自动识别为换行，换行时一定要空一行或者在换行的上一行打入两格以上的空格不然在remark解析.md文件时会将一个换行符解析为空格。
其中，空一行是会将上下两段文字解析成两个p标签，而打入两个空格再enter只会在显示上换行两段文字仍属于一个p标签内。  
&nbsp;

空行：markdown文件里和word不一样，无论打多少个换行符进去最后只会显示一个换行的效果，但有时我们想要段落之间的距离变大，就可以采用插入空白字符的方式顶掉下一行的位置  

```markdown
&nbsp; //空白字符
```
列表：markdown支持有序列表和无序列表。无序列表使用（*），（+），（-）作为列表标记，这些标记后加空格，而数字加点代表有序列表
而列表嵌套时只需在列表前多插入两个或四个空格就行


图片：开头一个感叹号 ! 接着一个方括号，里面放上图片的替代文字接着一个普通括号，里面放上图片的网址， 最后还可以用引号包住并加上选择性的 'title' 属性的文字。

链接：``` [链接名称](链接地址) 或者 <链接地址>```


代码块：当行代码块用单个单引号包裹代码，代码块使用三个反引号 ``` 包围，可以指定语言进行语法高亮。
&nbsp;  

ps:markdown是html的一个子集，所以部分可以使用html的书写格式
